# Open Source Finder

Open Source Finder helps developers discover GitHub projects that match their skills, interests, and contribution preferences.

This repository is organized as a Next.js frontend and FastAPI backend.

## Structure

- `frontend/` - chat-first discovery UI
- `backend/` - API, ranking pipeline, GitHub integrations, and persistence
- `scripts/` - local development and data utilities
- `docs/` - architecture, ranking, prompt, and UI notes
